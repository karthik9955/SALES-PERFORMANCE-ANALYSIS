# SALES-PERFORMANCE-ANALYSIS
Sales Performance Dashboard
📊 Project Overview

This project presents a Sales Performance Dashboard created using Power BI to analyze sales data and track business performance. The dashboard provides insights into sales trends, revenue generation, customer behavior, and key performance indicators (KPIs).

📁 Files Included
SALESPERFORMANCE.pbix – Power BI dashboard file
train(1).csv – Dataset used for analysis
🎯 Objectives
Analyze overall sales performance
Identify top-performing products and categories
Track sales trends over time
Monitor key business metrics
Support data-driven decision-making
📈 Dashboard Features
Total Sales Overview
Sales Trend Analysis
Product Performance Insights
Category-wise Sales Distribution
Interactive Filters and Slicers
KPI Cards and Visualizations
🛠️ Tools Used
Power BI
Microsoft Excel / CSV Data
Data Visualization
Data Cleaning & Transformation
🚀 How to Use
Download the repository.
Open SALESPERFORMANCE.pbix in Power BI Desktop.
Refresh the dataset if required.
Explore the interactive dashboard and insights.
📌 Key Insights
Sales performance can be monitored through interactive visualizations.
Top-performing products and categories are easily identified.
Trends and patterns help improve business strategies.
